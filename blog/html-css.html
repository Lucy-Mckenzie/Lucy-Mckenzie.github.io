What does it mean to understand inline, inline-block and block?

Understanding the differences between inline, inline block and block. display properties in CSS are crucial for controlling the layout of your HTML webpage. Think of these as different ways you can arrange items on a shelf.

By the end of this blog you should be able to customise your HTML using these properties.

Inline

First, we’ll start with inline. Inline displays an element as an inline element where any height and width properties have no effect. When something is displayed inline, it doesn’t push other items to the next line. Instead, it stays right where it is, alongside other items, just like text in a sentence.

Examples of inline elements:

  • <span>
  • <a>
  • <img>
  • <em>
  • <strong>
  • <i>
  • <small>

Inline-block

This is essentially the same thing as inline, except you have full control over height and width properties. This is useful when creating layouts where elements need to be inline but also require specific dimensions.

Block

Block elements start on a new line and take up the full width available. They do not allow other elements to sit on the same line. Imagine putting a big box on a shelf—everything else has to move to a new shelf because the box takes up the whole row.

Examples of block elements:

  • <div>
  • <li>
  • <p>
  • <section>

Analogy

Another way to understand inline, block, and inline-block is to imagine you're at a tree planting day in your local town with friends.

  • Inline: One friend plants her tree next to yours, but she has no control over the width or height. This is like planting inline.
  • Inline-block: Another friend, whose dad is running the event, picks how big or small her tree is but still plants it next to yours. This is inline-block.
  • Block: Your brother plants a tree that takes up the full width of the row, pushing everything else to the next row. This is block.

To Summarize

Inline, inline-block, and block are essential building blocks of CSS for design and development. They help you space, organize, and structure your HTML webpage effectively.
